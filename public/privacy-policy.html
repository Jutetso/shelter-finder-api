מדיניות פרטיות - Shelter Finder

עדכון אחרון: 2025-01-01

מידע שאנו אוספים

Shelter Finder אוספת מידע מינימלי הנדרש לפעולת האפליקציה:

  • מיקום: נדרש למציאת המקלט הקרוב. המיקום אינו נשמר בשרתים שלנו.
  • טוקן התראות: נדרש לשליחת התראות על אזעקות. אנונימי לחלוטין.
  • ערים נבחרות: נשמרות בשרת לצורך שליחת התראות רלוונטיות.
  • נתוני הגעה למקלט: נתונים אנונימיים על זמן הגעה למקלט לצורך שיפור השירות.

מידע שאיננו אוספים

  • שמות, כתובות דוא"ל, מספרי טלפון
  • מידע אישי מזהה
  • היסטוריית מיקום

שימוש במידע

המידע משמש אך ורק ל:

  • שליחת התראות על אזעקות בערים שבחרתם
  • מציאת המקלט הקרוב אליכם
  • שיפור השירות (נתונים אנונימיים בלבד)

אבטחת מידע

אנו משתמשים בהצפנת SSL ובשירותי ענן מאובטחים (Vercel, Firebase). הנתונים מוגנים בהתאם לסטנדרטים המקובלים בתעשייה.

שיתוף מידע עם צד שלישי

אנו לא מוכרים, מסחרים, או מעבירים מידע אישי לצדדים שלישיים. שירותי צד שלישי בשימוש:

  • Firebase Cloud Messaging — לשליחת התראות (Google)
  • Vercel — אחסון השרת
  • Pikud HaOref API — נתוני אזעקות

שמירת נתונים

טוקני התראות נשמרים כל עוד האפליקציה מותקנת. ניתן למחוק את הנתונים על ידי הסרת האפליקציה. נתוני הגעה למקלט נשמרים ב-Redis למשך 90 יום.

זכויות המשתמש

אתם רשאים בכל עת:

  • לבטל התראות דרך הגדרות האפליקציה
  • למחוק את המנוי על ידי הסרת האפליקציה
  • לפנות אלינו לשאלות בנוגע לנתוניכם

שינויים במדיניות

אנו עשויים לעדכן מדיניות זו מעת לעת. שינויים מהותיים יפורסמו באפליקציה.

יצירת קשר

לשאלות בנוגע למדיניות הפרטיות, ניתן לפנות אלינו דרך חנות האפליקציות.

Privacy Policy - Shelter Finder (English)

Last updated: 2025-01-01

Information We Collect

  • Location: Used to find the nearest shelter. Not stored on our servers.
  • Push notification token: Used to send rocket alert notifications. Fully anonymous.
  • Selected cities: Stored on server to send relevant alerts.
  • Shelter arrival data: Anonymous data on shelter response time to improve the service.

Information We Do NOT Collect

  • Names, email addresses, phone numbers
  • Personally identifiable information
  • Location history

Use of Information

Information is used solely to send rocket alerts for your selected cities, find the nearest shelter, and improve the service using anonymous data.

Third-Party Services

  • Firebase Cloud Messaging (Google) — push notifications
  • Vercel — server hosting
  • Pikud HaOref API — alert data

Contact

For privacy questions, contact us through the app store listing.
